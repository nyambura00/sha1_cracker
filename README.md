# SHA1 cracker
Implemented using Rust language.

## Usage

```
# sha1_cracker <wordlist.txt> <sha1_hash>
# Eg
$ sha1_cracker wordlist.txt 7c6a61c68ef8b9b6b061b28c348bc1ed7921cb53
```


## Run

```
$ cargo run -- wordlist.txt 7c6a61c68ef8b9b6b061b28c348bc1ed7921cb53
```


## Build

```
$ cargo build
```
